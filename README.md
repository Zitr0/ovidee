### 🇪🇸 Español

Ovidee es un editor de video agéntico, **open source y local-first**. Dos formas de empezar:

- **🎬 Arrastra tu metraje** — transcripción Whisper local, cortes con criterio editorial, transiciones, zooms, captions karaoke, títulos cinéticos y motion graphics por descripción libre.
- **🌐 Pega una URL** — captura el sitio con tu navegador local y genera un video del sitio con su propia identidad visual.

Apruebas el costo antes de gastar, iteras por chat, y la única API que necesitas es la de tu LLM.

<td width="50%" valign="top">

### 🇬🇧 English

Ovidee is an agentic video editor, **open source and local-first**. Two ways to start:

- **🎬 Drop your footage** — local Whisper transcription, editorial cuts, transitions, zooms, karaoke captions, kinetic titles, and free-form motion graphics.
- **🌐 Paste a URL** — captures the site with your local browser and generates a video of the site using its own visual identity.

You approve the cost before spending, iterate via chat, and the only API you need is your LLM's.

## ⚡ Quick start

```bash
git clone --recursive https://github.com/Zitr0/ovidee.git
cd ovidee
pnpm install
pnpm dev   # → http://127.0.0.1:3000
```
